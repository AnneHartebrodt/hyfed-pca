HyFed

HyFed (Hybrid Federated) is a framework for privacy-preserving machine learning. It provides an easy-to-use API to develop federated learning based algorithms, where each client (1) obtains the global parameters from the server, (2) computes the local parameters from its private data and global parameters, (3) perturbs the local parameters with noise, and (4) shares the noisy local parameters with the server and noise with the compensator. The compensator aggregates the noise values from the clients and sends the resulting aggregated noise to the server. The server aggregates the noisy local parameters and the negative of the aggregated noise to compute the global parameters.
