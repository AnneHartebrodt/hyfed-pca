Fever-PCA

Fever-PCA (Federated PCA for vertically partionned data) is a federated, privacy-preserving tool for Principal Component Analysis, including patient stratification and dimensionality reduction.

Unlike centralised PCA the data stays with the data owner and only model parameters are exchanged. For patient stratification, it is possible to compute the eigenvectors of the patient-by-patient covariance matrix, without exchanging the eigenvectors of this matrix. Only the eigenvectors of the feature-by-feature covariance matrix need to be exchanged.
